# ZERO

Nothing is there.

## Requirements

This theme requires:

* Movable Type 5.2 or higher

## Installing

+ Unpack the mt-theme-zero archive.
+ Upload and copy the mt-theme-zero/themes/zero folder to your MT's themes folder. ( /path/to/mt/themes )
+ Create a site with "Zero" theme.
